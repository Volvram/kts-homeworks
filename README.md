# Мобильное веб-приложение Crypto

Сервис для просмотра актуальной информации по курсам криптовалют.

Используется API coingecko.

Подробнее https://kts-crypto-currencies.netlify.app

# Getting Started

### `yarn dev`

# Start production

### `yarn build`

# Create production folder

### `yarn bundle`
